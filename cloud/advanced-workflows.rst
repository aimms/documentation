Advanced AIMMS Cloud Workflows
==============================

The basic Cloud workflow described in `Basic AIMMS Cloud Workflow <basic-workflow.html>`_ can be extended to suit the particular requirements of your application. This section describes the capabilities offered by the AIMMS PRO framework for setting up customized workflows. The underlying code and API are the same as for AIMMS PRO on-premises; the difference is that all portal UI interactions reference the Portal.
